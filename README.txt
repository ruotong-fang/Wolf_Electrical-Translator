EE Translator - Offline Edition

1. Double-click EETranslator.exe. Python is not required.
2. Quick Translation is intended for drafts. Professional Translation improves engineering style with the bundled local model.
3. After reviewing a translation, click "Remember Translation". The application stores sentence-level translation units and reuses exact or highly similar matches.
4. The built-in terminology is organized by electrical engineering domain. CSV import and export are supported.
5. TXT files support UTF-8, UTF-16, GB18030, Big5, and Windows ANSI. An encoding preview is shown when detection is uncertain.
6. All documents and translations remain on this computer.
